 澎湖陸上活動

 

 

澎湖是台灣最大的離島群，由90座島嶼組成。壯麗海洋美景和豐富人文歷史，讓澎湖成為國人最愛渡假勝地之一。 澎湖面積雖不大，認真玩起來景點美食卻不少；

一般遊客最愛澎湖三天兩夜行程，但要玩個透徹可能5天都不夠呢。

   

 澎湖自由行Day1：馬公市區、海洋牧場、澎湖花火節 抵達澎湖、租機車 / 汽車 → 中央老街 → 篤行十村 → 海洋牧場 → 澎湖花火節

 

 澎湖自由行Day2：澎湖南環一日遊 風櫃洞 → 蛇頭山 → 山水沙灘 → 隘門沙灘 → 奎壁山摩西分海 → 南寮古厝 → 夜釣小管

 

 澎湖自由行Day3：澎湖北環一日遊1 後寮天堂路 → 池東大菓葉玄武岩 → 三石壁 → 二崁聚落 → 池西岩瀑 → 小池角雙曲橋 → 漁翁島燈塔 → 三仙塔、外垵漁港 → 西嶼西臺

 

 澎湖自由行Day4：澎湖北環一日遊2 吉貝島 → 澎湖水族館 → 通樑古榕 → 跨海大橋 → 鯨魚洞、小門地質館 → 大義宮

 

 澎湖自由行Day5：七美、望安跳島一日遊 雙心石滬 → 小台灣 → 大獅風景區 → 七美人塚 → 網垵口沙灘 → 花宅聚落 → 天台山 → 買伴手禮、準備返台
